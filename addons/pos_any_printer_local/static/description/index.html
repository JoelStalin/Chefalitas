Conector de Impresora Local para TPV

Impresión directa desde Odoo a su impresora de recibos

Paso 1: Descargar el Agente de Impresión

Requerido para Windows

Este módulo requiere un pequeño agente de servicio en cada PC del TPV para comunicarse con las impresoras. Por favor, descargue e instale el agente en cada computadora que necesite imprimir recibos.

Descargar Agente de Impresión Local (Windows)

Paso 2: Configurar en Odoo

Una vez instalado el agente, vaya a TPV > Configuración > Puntos de Venta. Seleccione su punto de venta, active la opción "Activar Impresión Local (WebSocket)" y escriba el nombre exacto de su impresora.
